# AI Content Studio

AI-powered social media content platform for small and medium businesses. Users describe their business; the app generates product photos, graphics, video ads, captions, hashtags — and publishes directly to YouTube, TikTok, and Instagram.

## Status

Phase 0 — Scaffold in progress.

## Tech Stack

Next.js 15, TypeScript, Tailwind CSS, Clerk, Neon + Drizzle ORM, Muapi.ai, OpenAI, Stripe, Cloudflare R2.
